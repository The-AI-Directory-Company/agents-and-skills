# Go-to-Market Plan Template

Use this blank template to draft a go-to-market plan. Fill every section — if a field is unknown, mark it `[TBD]` with a deadline to resolve.

---

## 1. Positioning Statement

```
For [target customer]
who [need or pain point],
[Product name] is a [category]
that [key benefit].
Unlike [primary alternative],
we [key differentiator].
```

**Validation test:** Could a competitor copy this statement word-for-word? If yes, the differentiator is too generic.

## Quality Checklist

- [ ] Positioning cannot be reused verbatim by a competitor
- [ ] Segments prioritized with clear reasoning
- [ ] Channels mapped to segments with named owners and budgets
- [ ] Pricing context included and consistent with the GTM motion
- [ ] Timeline has phases, owners, and measurable milestones
- [ ] Every metric has a "below target" threshold and action plan
- [ ] Every channel and phase has a named owner (a person, not a team)
