# Platform Formatting Specs

Quick-reference constraints for each major platform. Always format posts to these specs before publishing.

## LinkedIn

| Attribute | Spec |
|---|---|
| Max post length | 3,000 characters |
| "See more" fold | ~210 characters (first 2-3 lines visible) |
| Media types | Single image, multi-image (up to 9, but 1 or carousel performs best), native video (up to 10 min), document/PDF carousel, polls |
| Image size | 1200x627px (link preview), 1080x1080px (square post), 1080x1350px (portrait) |
| Video specs | MP4, 3s-10min, max 5GB, 16:9 or 1:1 |
| Hashtag limit | 3-5 max (more than 5 reduces reach) |
| Link behavior | Links in post body reduce organic reach. Consider placing link in first comment. |
| Best post times (UTC) | Tue-Thu 13:00-15:00 |
| Algorithm notes | Dwell time (time reading) is a key signal. Longer posts that hold attention outperform short ones. Comments in the first hour matter most. |

## X / Twitter

| Attribute | Spec |
|---|---|
| Max post length | 280 characters (free), 25,000 characters (Premium) |
| Media types | Up to 4 images, 1 GIF, 1 video, polls |
| Image size | 1600x900px (16:9) or 1080x1080px (1:1). Max 5MB per image (PNG, JPEG, WEBP). |
| Video specs | MP4, max 2min 20s (free) / 4hr (Premium), max 512MB |
| Hashtag limit | 1-2 max (more than 2 reduces engagement measurably) |
| Thread format | Tweet 1 hooks, Tweet 2 context, Tweets 3-6 substance, final tweet CTA + self-reply with link |
| Best post times (UTC) | Mon-Fri 12:00-15:00 |
| Algorithm notes | Replies and quote posts weigh more than likes. First 30 minutes of engagement determine reach. Images increase impressions ~35%. |

## Instagram

| Attribute | Spec |
|---|---|
| Max caption length | 2,200 characters |
| Visible before "more" | ~125 characters |
| Media types | Single image, carousel (up to 20 slides), Reels (video up to 90s), Stories (15s segments) |
| Feed image size | 1080x1080px (square), 1080x1350px (portrait, recommended), 1080x566px (landscape) |
| Reel specs | 9:16 vertical, 1080x1920px, 3-90s, MP4, max 4GB |
| Hashtag limit | 5-15 in first comment (not in caption). Instagram's own guidance now suggests 3-5 highly relevant hashtags. |
| Link behavior | No clickable links in captions. Use "Link in bio" or Stories link sticker. |
| Best post times (UTC) | Mon/Wed/Fri 11:00-13:00 |
| Algorithm notes | Saves and shares weigh more than likes. Reels currently receive outsized distribution. Carousel posts get re-served when users interact with later slides. |

## TikTok

| Attribute | Spec |
|---|---|
| Max caption length | 2,200 characters (was 150, expanded in 2023) |
| Media types | Video only (photo mode converts to slideshow video) |
| Video specs | 9:16 vertical, 1080x1920px, 3s-10min (up to 30min for some accounts), MP4/MOV |
| Hashtag limit | 3-5 max |
| Link behavior | Link in bio (1,000+ followers). No clickable links in captions. |
| Best post times (UTC) | Tue/Thu/Sat 19:00-21:00 |
| Algorithm notes | Watch-through rate is the top signal. First 1-3 seconds determine whether the video gets distribution. Completion rate matters more than total views. |

## Cross-Platform Quick Reference

| Platform | Max Length | Media | Hashtags | Best Times (UTC) |
|---|---|---|---|---|
| LinkedIn | 3,000 chars | Image, video, PDF carousel | 3-5 | Tue-Thu 13:00-15:00 |
| X / Twitter | 280 chars | Image, GIF, video | 1-2 | Mon-Fri 12:00-15:00 |
| Instagram | 2,200 caption | Image, carousel, Reels | 3-5 (or up to 15 in comment) | Mon/Wed/Fri 11:00-13:00 |
| TikTok | 2,200 caption | Video only | 3-5 | Tue/Thu/Sat 19:00-21:00 |

## Notes

- All times are UTC. Adjust to your audience's primary timezone.
- Platform specs change frequently. Verify limits quarterly.
- "Best times" are industry averages. Your account's analytics may show different peaks — always prefer your own data over generic benchmarks.
