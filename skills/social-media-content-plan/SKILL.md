# Social Media Content Plan

## Before you start

Gather the following from the user:

1. **Which platforms?** (LinkedIn, X/Twitter, Instagram, TikTok, YouTube Shorts, Threads, Bluesky)
2. **What are the business goals?** (Brand awareness, lead generation, community building, product launches, hiring)
3. **Who is the target audience?** (Job titles, interests, pain points, which platforms they use)
4. **What is the posting cadence?** (Per platform — daily, 3x/week, weekly)
5. **What content assets exist?** (Blog posts, case studies, product screenshots, team photos, video clips)
6. **Brand voice guidelines?** (Tone, words to avoid, emoji usage, humor level)

If the user says "we need to post more on social," push back: "Posting more without platform-specific strategy just creates noise. Which platform drives the most value for your audience, and what outcome should it deliver this quarter?"

## Platform formatting specs

Each platform has different constraints. Always format posts to these specs.

```
| Platform       | Max Length     | Media          | Hashtags       | Best Post Times (UTC)     |
|----------------|---------------|----------------|----------------|---------------------------|
| LinkedIn       | 3,000 chars   | Image, video   | 3-5 max        | Tue-Thu 13:00-15:00       |
| X/Twitter      | 280 chars     | Image, video   | 1-2 max        | Mon-Fri 12:00-15:00       |
| Instagram      | 2,200 caption | Image, carousel| 5-15 in comment| Mon/Wed/Fri 11:00-13:00   |
| TikTok         | 2,200 caption | Video only     | 3-5 max        | Tue/Thu/Sat 19:00-21:00   |
| Threads        | 500 chars     | Image, video   | None           | Mon-Fri 12:00-14:00       |
```

## Content pillar mapping

Map each pillar to platform-specific formats. Not every pillar works on every platform.

```
| Pillar              | LinkedIn           | X/Twitter          | Instagram          |
|---------------------|--------------------|--------------------|---------------------|
| Thought leadership  | Long-form post     | Thread (5-8 tweets)| Carousel (8-10 slides)|
| Product updates     | Feature post+image | Single tweet+GIF   | Reel or story       |
| Customer stories    | Case study excerpt | Quote tweet format | Video testimonial   |
| Behind the scenes   | Team spotlight     | Casual observation | Photo/reel          |
| Educational         | How-to post        | Thread with tips   | Carousel tutorial   |
```

## Weekly content schedule template

Plan a full week per platform. Adapt the template to the user's cadence.

```
## Week of [DATE]

### LinkedIn (3x/week)
| Day | Post Type        | Topic                              | CTA            | Media       |
|-----|------------------|------------------------------------|----------------|-------------|
| Tue | Thought leader   | [Topic from pillar]                | Comment prompt  | None        |
| Thu | Customer story   | [Customer name] + result           | Link to case   | Screenshot  |
| Fri | Product update   | [Feature] launched this week       | Try it free    | Demo GIF    |

### X/Twitter (5x/week)
| Day | Format          | Hook (first 50 chars)              | Thread? | Media       |
|-----|-----------------|-------------------------------------|---------|-------------|
| Mon | Single tweet    | "Most teams get [X] wrong..."      | No      | None        |
| Tue | Thread (5 tweets)| "How we built [Y] in 2 weeks:"   | Yes     | Screenshots |
| Wed | Quote + opinion | RT industry news with take          | No      | None        |
| Thu | Tip tweet       | "Quick tip: [actionable advice]"   | No      | Image       |
| Fri | Engagement      | Poll or question to audience        | No      | None        |
```

## Post templates

### LinkedIn post template
```
[Hook — first 2 lines visible before "see more"]

[3-5 short paragraphs, each 1-2 sentences]

[Takeaway or lesson]

[Call to action — question or link]

[3-5 hashtags on final line]
```

### X/Twitter thread template
```
Tweet 1: [Bold claim or surprising stat — must stand alone]
Tweet 2: [Context — why this matters]
Tweet 3-6: [Steps, examples, or evidence]
Tweet 7: [Summary + CTA]
Tweet 8: [Self-reply with link or resource]
```

## Hashtag strategy

Build a hashtag bank per platform. Categorize by reach tier.

```
| Tier     | Volume       | Example               | Use            |
|----------|-------------|------------------------|----------------|
| Broad    | 1M+ posts   | #marketing #startup    | 1 per post     |
| Mid      | 100K-1M     | #contentmarketing      | 2-3 per post   |
| Niche    | 10K-100K    | #b2bsaas #devtools     | 2-3 per post   |
| Branded  | Custom      | #YourBrandName         | 1 per post     |
```

Never use more hashtags than the platform spec allows. On X/Twitter, more than 2 hashtags reduces engagement.

## Engagement schedule

Social media is not publish-and-forget. Define response windows.

```
Post published → Reply to comments within 2 hours of posting
Daily (15 min) → Engage with 5-10 posts from target audience accounts
Weekly (30 min) → Review top-performing posts, note patterns
Monthly         → Audit hashtag performance, retire underperformers
```

## Quality checklist

Before finalizing the content plan, verify:

- [ ] Every post has platform-specific formatting (length, media, hashtags)
- [ ] Post hooks are written — not just topics, but actual opening lines
- [ ] Each platform has a distinct content mix, not cross-posted identical content
- [ ] Hashtags are categorized by reach tier and within platform limits
- [ ] Engagement windows are defined with time commitments
- [ ] At least one week is fully drafted at the post level, not just themes
- [ ] CTAs vary across posts (not every post says "link in bio")

## Common mistakes

- **Cross-posting the same text everywhere.** A LinkedIn post is not a tweet. Rewrite for each platform's format, tone, and audience expectations.
- **Overloading hashtags.** Instagram tolerates 15; X/Twitter penalizes more than 2. Follow the platform spec, not a gut feeling.
- **Writing hooks that don't stop the scroll.** If the first line is "Excited to announce...", nobody clicks "see more." Lead with a surprising claim, a number, or a question.
- **No engagement plan.** Posting without replying to comments signals a broadcast account. Algorithms reward conversations, not broadcasts.
- **Ignoring platform analytics.** If carousel posts outperform single images on your account, make more carousels. Check performance weekly and adjust.
